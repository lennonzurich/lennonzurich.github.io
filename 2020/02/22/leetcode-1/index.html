Day1: two sum

1.暴力

  • Java

    执行用时 : 25 ms
内存消耗 : 44.8 MB

  • Python3
    ~Python3
    class Solution:

    def twoSum(self, nums: List[int], target: int) -> List[int]:
    for i in range(len(nums)):
        or j in range(i+1, len(nums)):
            if nums[i] + nums[j] == target:
                return [i, j]

MCN Introduction

Author: Lennon Jia Date: March 17, 2020  22:48:44 Category: 2020 Notes

MCN

定义：Multi-ChannelNetwork（多频道网络）

  • e.g. 跟网红签约的经纪公司

KOL: Key opinion leader

  • e.g. 网红

经营模式

  • MCN签约创作者

    • 提供资源支持
    • 跟平台接洽推广
    • 广告收入分成

MCN公司类型

  • 1.内容生产型

    • 有“内容”
    • 专有IP或专栏
    • e.g. 二更

  • 2.网红资源运营型

    • 签约大量网红
    • 聚焦泛娱乐领域
    • e.g.网红经纪公司

  • 3.自研孵化型

    • 自己打造素人
    • e.g. 大禹，papitube

  • 4.IP产业布局型

    • 通过品牌授权、衍生品开发等形式开展业务

    • 广为人知的形象IP

      • 版权变现

    • e.g. 漫画工作室

  • 5.电商内容型

    • 电商+内容
    • 短视频变现
    • e.g. 淘宝直播

地理分布

  • 北上广深最多，二线发达城市增长

基础业态

  • 内容生产业态

    • 以内容生产为主，逐步工业化，IP化
    • e.g. 二更

  • 运营业态

    • IP孵化

      • 内容

        • 内容策划
        • 选题把控
        • 内容传播

      • 平台

        • 规则研究
        • 内容分发

      • 账号

        • 账号定位
        • 粉丝管理
        • 矩阵规则

    • e.g. 大禹

      • 运营带动内容，ip孵化

    • e.g. 末那传媒

      • 人物形象ip+内容ip

  • 营销业态

    • e.g. 蜂群文化

      • 批量账号

        • KOL资料库

        • KOL全维度分析

          • 原创内容
          • 粉丝画像

        • 选择和campaign（风向？）契合度最高的KOL

        • 广告主，博主沟通

        • 确认KOL名单和推广方案

  • 电商业态

    • 红人电商

      • 红人直接带动销售转化
      • e.g. 张大奕，学历，滕雨佳，李佳琪

    • 内容电商

      • e.g. 李子柒

  • 经济业态

    • e.g. papitube

      • 孵化+经纪+营销

        • 商业全约
        • 艺人约
        • 商务约

  • 社群/知识付费业态

    • e.g. 灵魂有香气的女子

      • 内容+电商+广告+社群

  • IP授权，版权业态

    • e.g. 幕星社

变现方式

  • 第一梯队

    • 广告营销：80%

      • e.g. 大禹文化，橘子娱乐，万粒科技

    • 平台补贴：48%

  • 第二梯队

    • 内容电商：35.5%

      • e.g. 如涵控股，微念科技，美ONE

    • 课程销售：22.6%

  • 第三梯队

    • 衍生品销售

    • ip授权

      • e.g. 吾皇的白茶，长草颜团子

平台依赖

  • 社交平台

    • e.g. 微信，微博
    • KOL资源池，具备媒体树形，社交沉淀优势明显

  • 资讯平台

    • 算法推荐，精准匹配受众
    • 流量优势

  • 短视频平台

    • 强大的生产工具
    • 用户下沉覆盖三四线城市
    • 大量UGC-用户原创内容

  • 垂直平台

    • e.g. bilibili，马蜂窝，汽车之家
    • 用户特征突出，社群粉丝效应明显
    • 影响力大

  • 电商平台

    • 用户购买转化率高

  • 直播平台

    • 付费意愿直接
    • 红人表现直接
    • 利于粉丝运营
